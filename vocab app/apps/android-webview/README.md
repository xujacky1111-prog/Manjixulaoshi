# Android WebView Wrapper

Open this folder with Android Studio and build the `app` module.

The app opens:

```text
https://layer-city.com/word/
```

To point it at another deployment, edit:

```text
app/src/main/res/values/strings.xml
```

This is a WebView wrapper, so the server, database, AI settings, and token monitor still run on the hosted PHP app.
