# Word Bank Data

`high_school_starter.csv` is a small starter bank used to test the new cloud-download and local-APK flow.

For the full high-school bank, import a reviewed 3000-3500 word list into `/wordapp/vocabbase/` using:

```text
word | pos | meaning_zh | example_en
```

The Android app will then download it through `/wordapp/api/word-bank.php?code=high_school`.
